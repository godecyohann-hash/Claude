# 🥐 AMANN PÂTISSERIE — App

## Comment déployer sur iPhone (étapes simples)

### Étape 1 — Créer un compte GitHub (gratuit)
1. Allez sur **github.com**
2. Cliquez "Sign up" et créez un compte gratuit

### Étape 2 — Uploader ce dossier sur GitHub
1. Sur GitHub, cliquez le bouton vert **"New"** pour créer un nouveau repository
2. Nommez-le `amann-patisserie`
3. Cliquez **"uploading an existing file"**
4. Glissez-déposez **tous les fichiers** de ce dossier
5. Cliquez **"Commit changes"**

### Étape 3 — Déployer sur Vercel (gratuit)
1. Allez sur **vercel.com**
2. Cliquez "Sign up" → connectez-vous avec votre compte GitHub
3. Cliquez **"New Project"**
4. Sélectionnez votre repository `amann-patisserie`
5. Cliquez **"Deploy"** — attendez 2 minutes
6. Vercel vous donne une URL comme : `https://amann-patisserie.vercel.app`

### Étape 4 — Installer sur iPhone
**Pour vous (Admin) et vos pâtissiers :**
1. Ouvrez **Safari** sur iPhone
2. Allez sur votre URL Vercel
3. Appuyez sur le bouton **Partager** (le carré avec une flèche ↑)
4. Faites défiler et appuyez **"Sur l'écran d'accueil"**
5. Appuyez **"Ajouter"**

✅ L'app apparaît sur votre écran d'accueil comme une vraie application !
